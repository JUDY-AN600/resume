# 安喆个人网站

## 预览
直接双击 `index.html`。

## 修改内容
网站的大部分文字都在 `content.js` 中：
- `profile`：封面信息
- `categories`：项目分类
- `projects`：项目卡片与详情
- `timeline`：完整经历
- `education`：教育经历
- `skills`：技能

## 替换照片
把新照片放入本文件夹，并在 `content.js` 中修改：
```js
photo: '你的照片文件名.jpg'
```

## 新增项目
复制 `projects` 中任意一组 `{ ... }`，修改文字即可。`category` 必须与分类 ID 对应。

## 样式修改
颜色、字号、卡片尺寸和手机端布局都在 `styles.css` 中。

## 发布
可将整个文件夹上传至 GitHub Pages、Netlify 或 Vercel。网站不依赖构建工具。


## 背景色变化
项目与经历区域的背景色均由滚动位置自动触发，不提供手动切换按钮。经历区从项目区最后一张卡片的色调继续变化，保持视觉连贯。


## 颜色过渡
项目区第一种颜色与封面一致；经历区承接项目区最后一种颜色，并在最后一项回到联系页的纸张底色，避免章节交界处突兀。
